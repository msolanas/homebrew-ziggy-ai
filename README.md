# homebrew-ziggy-ai

Homebrew tap for [ZiggyAI](https://github.com/q6cvy7kyxj-droid/ZiggyLocalAI) — a macOS menu bar app for local Gemma 4 inference via MLX.

## Install

```bash
brew tap q6cvy7kyxj-droid/ziggy-ai
brew install ziggy-ai
```

## Setup

Download the AI model (~15GB):

```bash
ziggy-ai setup
```

## Usage

```bash
ziggy-ai start          # Start the inference engine
ziggy-ai stop           # Stop the engine
ziggy-ai restart        # Restart the engine
ziggy-ai status         # Show engine status
```
